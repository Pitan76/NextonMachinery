# Nexton Machinery (English)
A mod introduces various machines and processing units used for automation, and resource transformation. It expands gameplay with industrial-scale production systems.<br />
Forked from Electrical Industry (by the same original author)

[//]: # (- CurseForge: https://www.curseforge.com/minecraft/mc-mods/nexton-machinery)
[//]: # (- Modrinth: https://modrinth.com/mod/nexton-machinery)

## Libraries
- Fabric API
- MCPitanLib
- Reborn Energy API (included internally)

## Blocks
- Generator - A machine that generates energy from fuel sources, such as coal or lava.
- Electric Furnace - A machine that smelts items using electricity instead of fuel.

# Nexton Machinery (日本語)
このmodは、オートメーション、資源変換のためのさまざまな機械や処理ユニットを導入します。産業規模の生産システムでゲームプレイを拡張します。<br />
Electrical Industryからフォークされました （同様のオリジナル作者によるもの）

## 前提Mod
- Fabric API
- MCPitanLib
- Reborn Energy API (内部に埋込済み)

## ブロック
- 発電機 - 石炭や溶岩などの燃料源からエネルギーを生成する機械。
- 電気炉 - 燃料の代わりに電気を使用してアイテムを精錬する機械。
